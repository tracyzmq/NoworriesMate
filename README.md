# NoworriesMate - 反内耗小助手

一款帮助用户减轻心理压力、减少精神内耗的微信小程序。通过有趣的清单勾选功能，生成个性化的“功德报告”，配合个人中心和历史记录功能，帮助用户记录和追踪自己的心理健康状态。

## 功能特色

- **反内耗清单**：提供多种缓解内耗的日常小任务，用户可勾选完成
- **功德报告生成**：将勾选的内容生成精美的图片报告
- **个人中心**：展示用户统计数据和个人信息
- **历史记录**：保存用户每天的勾选记录，形成打卡习惯
- **每日语录**：随机展示减压、心理健康的正能量语句

## 主要页面

- 首页：反内耗清单勾选和报告生成
- 个人中心：用户信息和统计数据
- 历史记录：过往打卡记录查看和管理

## 技术特点

- 微信原生小程序开发
- Canvas绘图生成报告图片
- 本地存储管理用户数据
- 响应式设计适配不同屏幕尺寸
